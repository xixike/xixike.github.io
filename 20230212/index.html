# B. 连通性（floyd）

挺复杂的计数题。

50 分应该还是比较好拿的，n=mn = mn=m 的情况就是贝尔数第 nnn 项，也即第二类斯特林数前缀和。

下面说正解。

观察 Floyd\text{Floyd}Floyd 运行机制，我们设前 n−mn - mn−m 个点为黑点，后 mmm 个点为白点。

那么合法的图一定是下面两种情况之一：

  • 黑白点都有的连通块，对于任意点对 (u,v)(u, v)(u,v) 都能找到一条路径满足除端点以外都是黑点的路径。

  • 都是白点的连通块，一定是完全图。

也就是说如果一个连通块里有黑点，那么黑点的导出子图必须连通，且每个白点至少与一个黑点相连。

然后就可以 dp\text{dp}dp 了。

# C. 文本串的价值（value）

把 AC 自动机建出来。

先说暴力 dp\text{dp}dp，设 fi,j,Sf_{i, j, S}fi,j,S​ 表示匹配到文本串 sss 的第 iii 个字符，在 trie\text{trie}trie 图上的 jjj 号点，当前填了状态为 SSS 的字符。

转移时判断 sss 的下一位是不是问号，是的话枚举字符，不是的话直接跳。

注意：你要记录模式串的贡献时要在 trie 图上做链和（根到当前点的链和）。

不是子树和！！！不是子树和！！！不是子树和！！！

然后你就能拿到 30 分的好成绩。

观察到如果 si+1s_{i + 1}si+1​ 不是问号，转移是固定的，我们还要枚举实在是浪费。

并且最多只有 14 个问号。

所以我们预处理出来 dpi,jdp_{i, j}dpi,j​ 表示在第 iii 个问号且当前在 trie 图上 jjj 号点，依次跳每个字符到 i+1i + 1i+1 个问号之前会到达哪个点，且中间会得到多少收益。

这样一来，我们转移的时候可以先枚举状态 SSS，计算出当前是在哪两个问号之间跳，然后枚举转移字符 ccc，当前位置 jjj 就可以转移了。

时间复杂度 O(214∣T∣)O(2^{14}|T|)O(214∣T∣)，有个字符集的常熟。

感觉正解比暴力好写。
