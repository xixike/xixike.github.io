# 主要思想

WQS 二分也叫带权二分，主要解决一类凸包上的问题。

假设当前你有一个很难计算的凸函数 f(x)f(x)f(x)，此时你要计算一个指定的 f(n)f(n)f(n)，如何快速求出呢？

我们可以去二分一个斜率 kkk，然后计算斜率为 kkk 的一条直线落到这个凸壳上时的交点。

设这个交点为 (x,f(x))(x, f(x))(x,f(x))，再设函数 g(k)g(k)g(k) 表示表示斜率为 kkk 的直线落到凸壳上时的截距。

那么：

f(x)=g(k)+kxg(k)=f(x)−kxf(x) = g(k) + kx\\ g(k) = f(x) - kxf(x)=g(k)+kxg(k)=f(x)−kx

由于 g(k)g(k)g(k) 是一条直线，感性理解一下，我们在计算这条直线上的值时就不需要考虑原题中的限制条件了。

于是我们就可以通过二分斜率来快速的找出答案了。

可以通过下面的例题来理解一下。

# 例题

# Description

P4383 [八省联考 2018] 林克卡特树

# Solution

首先转化一下题意，删边再加边显然是我们不想看到的，所以我们可以理解为从这棵树中选择 k+1k + 1k+1 条不相交的链，使它们的权值和最大。

我们先推出 60pts 的暴力 dp。

设 dp[0/1/2][x][i]dp[0/1/2][x][i]dp[0/1/2][x][i] 表示以 xxx 为根的子树中选择 iii 条完整链，xxx 点的度数分别为 0/1/20/1/20/1/2 时的最大权值和。

然后分 3 种情况讨论：

  • xxx 点度数为 0，那么此时 xxx 不在链上，于是合并一下子树的权值即可。

    dp[0][x][i]=max⁡(dp[0][x][j]+dp[0][y][i−j])dp[0][x][i] = \max(dp[0][x][j] + dp[0][y][i - j])dp[0][x][i]=max(dp[0][x][j]+dp[0][y][i−j])

  • xxx 点度数为 1，xxx 可能本身是一条链上的端点，或者当前从儿子连上来一条边，如果连上来一条边的话要记得加上边权。

    dp[1][x][i]=max⁡(dp[1][x][j]+dp[0][y][i−j],dp[0][x][j]+dp[1][y][i−j]+wx,y)dp[1][x][i] = \max(dp[1][x][j] + dp[0][y][i - j], dp[0][x][j] + dp[1][y][i - j] + w_{x, y} )dp[1][x][i]=max(dp[1][x][j]+dp[0][y][i−j],dp[0][x][j]+dp[1][y][i−j]+wx,y​)

  • xxx 点度数为 2，这时也会有两种情况，xxx 本身就是在一条链的中间，或者 xxx 本身是一条链的端点，yyy 也是一条链的端点，然后把 xxx 和 yyy 连到一起，注意也要加上边权。

    dp[2][x][i]=max⁡(dp[2][x][j]+dp[0][y][i−j],dp[1][x][j]+dp[1][y][i−j]+wx,y)dp[2][x][i] = \max(dp[2][x][j] + dp[0][y][i - j], dp[1][x][j] + dp[1][y][i - j] + w_{x, y} )dp[2][x][i]=max(dp[2][x][j]+dp[0][y][i−j],dp[1][x][j]+dp[1][y][i−j]+wx,y​)

最后合并：

dp[0][x][i]=max⁡(dp[1][x][i−1],dp[2][x][i])dp[0][x][i] = \max(dp[1][x][i - 1], dp[2][x][i])dp[0][x][i]=max(dp[1][x][i−1],dp[2][x][i])

答案就是：dp[0][1][k]dp[0][1][k]dp[0][1][k]

这样的复杂度是 O(nk2)O(nk^2)O(nk2) 的，显然无法通过此题，考虑优化。

不难发现，dpdpdp 值是一个上凸函数，这时 WQS 二分就登场了。

我们二分一下斜率 midmidmid，每次暴力计算一下斜率为 midmidmid 的直线的值，这时没有了 kkk 条边的限制，我们可以 O(n)O(n)O(n) 计算出 dpdpdp 值。

然后计算切点的位置，如果在 kkk 左边，斜率变小，反之斜率变大。

于是这题就可以在 O(nlog⁡k)O(n\log k)O(nlogk) 的复杂度内完成啦。
