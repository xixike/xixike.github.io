# P5283 [十二省联考 2019] 异或粽子

一道经典题目，介绍一个比较简单的做法。

先做一个前缀异或和，题目就变成了找到 si⊕sjs_i \oplus s_jsi​⊕sj​ 前 kkk 大的点对，且 i<ji < ji<j。

i<ji < ji<j 的条件并不好处理，所以把这个上三角形对称一下，变成完整的矩形，然后就变成了找到前 2×k2\times k2×k 大的点对。

具体操作时，建一个大根堆，对于第 iii 行，把与其异或最大值插到堆中。

每次删除堆顶最大值，假设堆顶是第 iii 行的第 ccc 大值，那么就把第 iii 行的第 c+1c + 1c+1 大值再插入堆中。

以此类推，找到前 2k2k2k 个点对即可，最后答案再除以 2。

时间复杂度 O((n+k)log⁡alog⁡n)O((n + k)\log a\log n)O((n+k)logalogn)。

# CF241B Friends

是上面那道题加强版。

考虑先把第 kkk 大值找到。那么二分第 kkk 大值，枚举每一个数，在 01trie01trie01trie 上找到与当前数异或值大于等于 midmidmid 的个数，最后跟 2k2k2k 比较大小即可。

这部分时间复杂度 O(nlog⁡2a)O(n\log^2a)O(nlog2a)，也可以直接 01trie01trie01trie 上二分，但是我不会。

然后要计算所有异或值大于等于 kthkthkth 的和是多少。

先预处理一个数组 tr[x][y]tr[x][y]tr[x][y]，表示在 01trie01trie01trie 上 xxx 的子树中 2y2^y2y 出现了多少次。这样就可以在 O(log⁡a)O(\log a)O(loga) 的复杂度下计算一棵子树异或一个值的和是多少了。

这部分时间复杂度也是 O(nlog⁡2a)O(n\log^2a)O(nlog2a)。

小坑点：2×k2\times k2×k 会爆 intintint，所以二分 checkcheckcheck 时需要开 long longlong \ longlong long。
