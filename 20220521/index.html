树状数组，dp + 矩阵乘法

# A. 空

O(n2)O(n^2)O(n2) 暴力就不多说了。

分两种情况讨论。

  • 线段 i,ji, ji,j 不包含，且 iii 在 jjj 右边，线段异或值为 (ri−lj)−(rj−li)(r_i - l_j) - (r_j - l_i)(ri​−lj​)−(rj​−li​)，开括号得 (li+ri)−(lj+rj)(l_i + r_i) - (l_j + r_j)(li​+ri​)−(lj​+rj​)，这里 li+ril_i + r_ili​+ri​ 是定值，如果我们想让整体最大，那么 lj+rjl_j + r_jlj​+rj​ 最小，树状数组维护即可。
  • 线段 jjj 包含线段 iii，那么异或值为 (rj−lj)−(ri−li)(r_j - l_j) - (r_i - l_i)(rj​−lj​)−(ri​−li​)，要使异或值最大，那么 rj−ljr_j - l_jrj​−lj​ 最大，同样使用树状数组维护即可。

时间复杂度 O(nlog⁡n)O(n\log n)O(nlogn)

# B. 早苗

套路的设 fi,jf_{i, j}fi,j​ 表示到了第 iii 天，连续 jjj 天刮不同的风的方案数。

转移方程：

fi+1,j+1+=fi,j×(m−j)fi+1,k+=fi,j(k∈[1,j])f_{i + 1, j + 1} += f_{i, j} \times (m - j) \\ f_{i + 1, k} += f_{i, j}(k \in [1, j]) fi+1,j+1​+=fi,j​×(m−j)fi+1,k​+=fi,j​(k∈[1,j])
