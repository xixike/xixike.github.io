# Day -1

今天有 tbl 学长的课，害怕.jpg

上午不出意外的被 D 了 QwQ

可能是因为在二南的时候听过了一次 tbl 学长的课，所以这次听的时候有了点心理准备，受到的折磨没有想象中的那么大。

晚上大家一起在机房打了一场 ABC，结果集体降智，一个小时硬是没切掉 F，自闭了。

好不容易挺过了今天，晚上回家调了调网络流洗了个澡就睡了。

# Day 0

上午简单收了收东西，由于不用带被子被褥什么的，所以也没有什么需要带的。

吃午饭的时候看了男子冰球决赛，俄罗斯奥委会对战芬兰，看的时候已经 1​ : 1​ 了，本来还等着看无限加时赛呢，结果芬兰第三节不到 1 分钟就进了一个球，于是后面的剧情就可想而知了。

比赛结束之后发现这是本届冬奥会最后一个项目，恭喜芬兰夺金！

然后就到点了，出去做地铁到火车站，碰到了 shukuangshukuangshukuang，结果都到检票点了 lj 还没有来，于是我们就先进去了。

到了车上，先小睡了一会，然而也就睡了不到半个小时，然后起来开颓！

打了一把 SkulTheHeroSlayerSkul\ The\ Hero SlayerSkul The HeroSlayer，发挥不太好，被 boss 秒掉了，于是觉得没啥意思，准备水洛谷。

由于从来没带过手机，并没有意识到自己可以开热点，于是点开了网络，发现了名叫 zxakioi 的热点，盲猜了一波密码就连上了。

后来发现热点是 sleeping_crawlerssleeping\_crawlerssleeping_crawlers 的，结果她以为我们在卷，把热点关掉了，这可让我怎么颓 /fn/fn

所以就打开了自己的热点，随便贺了一道昨天 tbl 学长刚讲过的题，然后继续开摆。

晚饭也没有吃，随便啃了两包薯片就不吃了，其实还有零食，不过在箱子里，不方便拿，就没吃。

9:30 车到站了，然后我和 AcestarAcestarAcestar 以及 shukuangshukuangshukuang 坐了同一辆出租车，然后奇妙的出租车之旅就开始了。

司机阿姨疯狂搭话，然而我们一个字都听不懂？！

边开车边刷淘宝，开着开着车不知道自己是在直行道还是左转道？

平均时速 95 迈，遇到黄灯不多说直接一脚油门冲过去？

非常庆幸一路上没出什么事……

有惊无险的到了宾馆 —— 锦江之星。在前台登记了半天过了好久才进屋。

感觉环境还可以。但是！刚要坐到床上的我突然发现了一个问题！

“嗯？这两张床怎么不一样大？？”

于是和 AcestarAcestarAcestar 进行了一波紧张刺激的石头剪刀布，然后我就赢了哈哈哈。

房间内的空调自动就打开了，喷出来的风还挺热的，常州也比想象中的暖和。

# Day 1

早上 6:45 起床，7:00 左右下楼吃饭，随便搞了点东西就吃了。

然后走到常中，路上看见了许多有着浓郁南方特色的小楼，感觉跟北方就是不一样。

到了机房，已经做好了爆零准备的我丝毫不慌。

开考了，迅速花费了不到 1 个小时的时间把每道题都看了看想了想，发现自己只会 T1 的 20pts 的暴力分，所以就先打上了。

然后觉得可以优化，就又开了个 cpp 文件，把我的暴力代码复制进去，然后在原 cpp 文件里改，丝毫没有意识到问题的严重性。

优化了一会发现想假了，就先放着了。

接着去看 T2，计算几何，感觉就是比较麻烦，于是去写 m≤4m \leq 4m≤4 的部分分，手动计算每一个对称点，写了快 2h，感觉还挺稳的。

然后又去写 n=4n = 4n=4 的分，在考试结束前调完了，迅速建好文件夹，就等着收代码了。

最后 5 分钟的时候随手点开了 T1 的代码，我：“？？？这都是什么玩意？”

赶紧去找之前的代码，发现那个文件已经被我删掉了。迅速手动改，程序里有一段代码需要写两遍，所以我就复制了一遍，然后…… 然后就有一个变量名忘改了，而且小样例里那个变量的值还是 0，过了小样例的我感觉世界平和了……

出分了，发现我过然还是垫底了 QwQ

# Day 2

吸取了第一天教训的我励志不要再爆零。

然而历史总是惊人的相似，T3 写了 30pts 部分分的我成功再次挂到了 5pts，我甚至现在还不知道哪里写挂了。

下午放了正睿的课，然而高二的学长们已经听过了。然后把 A 题改了。

吃完晚饭之后和 AcestarAcestarAcestar，ShukuangShukuangShukuang，CoudlySkyCoudlySkyCoudlySky 以及 sleeping_crawlerssleeping\_crawlerssleeping_crawlers 在校园里面转了转，发现好多打球散步的，感觉这里好悠闲，实名羡慕了。

晚自习很快就过去了，晚上回到房间去洗了个澡，这水量温度可比家里的好多了……

# Day 3

早上准点起床，去餐厅迅速吃完早饭，再次走到了常中机房。

开题，看题。

先把 T1 爆搜打了，然后觉得可以 O(n2)O(n^2)O(n2) dp，但想了半天不会转移，就先跳了。

接着看 T2，一眼莫队套线段树，复杂度 O(nnlog⁡n)O(n\sqrt n\log n)O(nn​logn)，期望得分 70pts，直接开写，花了不到 1h 写完调完，过完中样例，感觉稳了（此时的我依旧没有意识到事情的严重性）。

最后开 T3，看了两眼就到吃饭点了，于是先去吃饭。

交流了一下发现人均会 T1 O(n2)O(n^2)O(n2) 做法，很慌，然而慌并没有什么卵用。

吃完饭继续写，感觉这 T3 不是给人写的，写了个大爆搜就跑路了。

下午出分了，我 T1 怎么爆零了！我 T2 怎么就 35pts！！

然后发现 T1 文件名写成 T3 的了，T2 莫名被卡常了，可能数据故意卡莫队 QwQ

youwikeyouwikeyouwike 或成为全场唯一切掉 T3 的人，赶紧膜拜，然后去看他代码。发现他是瞎爆搜，只要两个相邻的格子颜色不一样就合法，然后就过了……

经验证，发现是 T3 的 spj 挂了，真就不 T 就能 A 呗。

晚上有一场 CF 阳间场，6:05 到 8:05，我们组团打 div.2 ，高二学长们打 div.1，正好他们能跟我们有重题。

延时了 5 分钟，不知道为啥。

开 A，卡了 3min 题面终于出来了，结果是个阅读理解，这对中国选手太不友好了，经过 IgnotusIgnotusIgnotus 不懈的翻译之后我们终于得到了准确的题意。

赶紧写，然后就过了。接着去看 B 题，sleeping_crawlerssleeping\_crawlerssleeping_crawlers 一语道破结论，太强了 orz，于是也迅速通过了。

C 题是个简单题，直接暴力删数就完了（然后真的就完了……）

D 题是巨大构造，想了好久没啥明确的思路，llllll 学长给我们讲了讲于是我们就知道构造方法了，然而代码比想象中的难写，调了快 30min 才通过。

比赛结束之后不久……

youwikeyouwikeyouwike：“我 C fst 了！”

AcestarAcestarAcestar：“我也是！”

我赶紧看了一眼，发现我也 fst 了！！然后发现是因为没开 longlonelong\ lonelong lone QwQ

最后全场就 sleeping_crawlerssleeping\_crawlerssleeping_crawlers 一个人没 fst CCCCCCCOrz

# Day 4

早上起来，发现他们都看了昨天 cf 上了多少分，我给忘了看了，于是到了机房之后打开 cf 主页看了一下，涨了 145 分，感觉还好，本来想着不挂分就行。

发现这是我第一次进前 1000，可能是因为同时有 div.1 和 div.2，大佬们都去打 div.1 了。

今天的模拟赛再次爆零了 QwQ，看了 T1 感觉暴力非常简单，想着拿莫队线段树什么的乱搞一下，发现不太行，因为 dp 值没法新加节点进去，打完暴力之后果断放弃。

去看 T2，怎么又是期望啊，我是真的不会算这种东西，然后就一直在调爆搜，考完了都没调出来 QwQ。

中间去看了 T3，感觉这又是不给人做的题了，n≤3n \leq 3n≤3 的分我都不会，爬了。

出分了，果不其然的垫底了 /kk

晚上回宾馆，发现母上大人让带的橘子都快要烂了，有几个已经出水了，于是赶紧吃了几个。

本来还打算把 w8 给的魔鬼花生吃了，结果 ShuKuangShuKuangShuKuang 突然跑到我们屋要了个橘子，说是 w8 给的花生太辣了，受不了了。

于是我们立马放弃吃花生的打算。

# Day 5

早上开题，看题。

感觉 T1 就暴力合并一下子能得不少分，于是写了个类似于 kruskalkruskalkruskal 的算法，瞎搞了搞就把大样例过了（大样例比较小）。

然后去看 T2，看了一眼数据范围发现数据里有各种奇奇怪怪的性质，于是一顿乱写 + 数据分治。不到 11 点写了个能得 35pts 的代码。

最后开 T3，感觉和 Day1 的 T3 有点类似，然而根本不会，于是随后敲了个 a1=1a_1 = 1a1​=1 的 10 分部分分，此时还不到 11:30

中午吃完饭回来，感觉自己没啥可写的了，所以就趴在桌子上睡觉。

距离比赛还有 15 分钟的时候醒了，检查了一下头文件，建了子文件夹，顺便检查一下代码，测了一下手造小样例。

嗯…… 等等！我 T2 样例没过是怎么回事？

赶紧去看自己的大爆搜，发现自己判断虫洞是否正确瞎写了，赶紧改。还有 5 分钟的时候总算是改对了。

然后…… 诶，这个爆搜代码会把一些不必要的虫洞输出出来，应该没问题吧？

等等！题面说了要把所有的点都匹配上虫洞…… 都匹配上…… 那完了，那我部分分都只输出了必须的虫洞，剩下的点都没匹配，那岂不是 1 分都没有了？！

然而已经来不及改了…… 算了，就这样吧 QwQ

出分了，发现自己有爆搜分和 m=2×n+1m = 2 \times n + 1m=2×n+1 的部分分…… 还好爆搜最后是调对了，不然又要爆零了 QwQ

下午讲题，T1 整几个堆然后合并就完了。T2 神仙构造，大力分讨，T3 集训队爷 yy 也不会了 /kk，所以没讲。

下午听了字符串，感觉海星。

商量了一下决定还是 27 号回去，于是告诉老师准备改签了。

晚上 9:30 下课之后 w8 和 jimmy 带着我们去做核酸，走了 114514 分钟终于到了医院，然后又等了 114514 分钟 jimmy 去大厅交钱终于能做核酸了。

回宾馆的时候已经快 11:00 了。

感觉这个宾馆天天把用过哪怕就用了那么一下的东西都换个新的还是挺离谱的，比如说梳子……

另外这个空调也是离谱，被我拿着遥控器一顿乱点之后就不出风了，出来的还是不怎么热的，没办法了晚上好好盖着被子睡觉吧。

# Day 6

w8 说常中今天晚半个小时比赛，因为是周六，所以我们把闹钟也调晚了半个小时，结果不知道为啥 AcestarAcestarAcestar 把铃声给调没了……

早上根本没闹钟！还好 ShuKuangShuKuangShuKuang 在外面敲了半天门直接给我们敲醒了 CCCCCOrz

到了机房，想着这是最后一场比赛了怎么不也得好好摆打吧。

然而事实上根本没给我选择的机会，这题一道都不会你让我怎么打？暴力也没一个能写的。

中途我们好几个人去厕所吹了半天水，交流了一下，发现大家也是啥都不会。

吃饭前把 T1 的 10pts 的 O(3nn2)O(3^nn^2)O(3nn2) 巨大暴力写了，然后就一直再看动态 dp，板子了，因为实在是没法写了。

中午吃完饭，ShuKuangShuKuangShuKuang 题意在校园里面转一转，反正啥也不会了，想了想感觉很对，于是又叫上了 CloudySkyCloudySkyCloudySky 以及 JohnsonloyJohnsonloyJohnsonloy 一块转了会。

感觉常州突然变得很暖和，可能是因为出太阳了吧。

回机房之后感觉自己的爆搜跑的有点慢，所以准备优化一下判是否合法的方法，从 O(n2)O(n^2)O(n2) 优化到 O(n)O(n)O(n)，此时是 12:00，两个小时你能秒我？

结果调到比赛结束还是没调出来 QwQ，没办法，只好交之前的暴力了。

啪的一下，很快啊，它就评测完了，非常高兴看到我成功得了 10pts，成功没有垫底，后面还有一大车爆零的，大家都开始打假赛了 QwQ

晚上有一场 ABC，时间上正好可以打，于是大家又开始了 ACM。

切 A 切 B，读 C，感觉 C 有点麻烦，就去读 D 了。

发现 D 就是个开个 set 然后暴力查找，于是码码码，结果边界一直调不对，非常自闭。

IgnotusIgnotusIgnotus 遗憾 -1，发现得先插入一个无穷大和一个无穷小，然后 AcestarAcestarAcestar 直接粘了一发平衡树代码就过了，比我们调 set 过得还快 QwQ

我又调了调也过了。

此时先写 C 再写 D 的 youwikeyouwikeyouwike 也已经切掉 C 了。

我们写 C 各种 WA，最后除了 youwikeyouwikeyouwike 一遍过其他人都有罚时。

接着去看 E，一眼找循环节暴力计算，然后开始码代码，大家陆陆续续的也都过了。

看完 F 之后发现是一个毒瘤用 map 存图然后跑最短路，而且建图各种判断巨麻烦，然而此时距离比赛结束只有十几分钟了，所以大家就都开始摆了。

晚上回宾馆之后收拾了一下行李，把羽绒服塞到箱子里了。

刚开始还怕不穿羽绒服会冷，然而事实上一点都不冷。
