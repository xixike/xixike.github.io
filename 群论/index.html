# Pólya 定理

考虑一种置换会形成许多置换环。

设 h(f)h(f)h(f) 为置换 fff 的置换环数量，若要用 kkk 种颜色给集合 XXX 染色，那么有 c(f)=kh(f)c(f) = k^{h(f)}c(f)=kh(f)。

证明：

# [POJ 1286] Necklace of Beads

题目描述

nnn 个珠子串成一个圆，用 3 种颜色去涂色。问一共有多少种不同的涂色方法。（旋转和对称同构）

题解

旋转同构和对称同构分开考虑。

  • 旋转：跟上面那道题是一样的，甚至直接暴力计算 gcd⁡\gcdgcd 都能过。

  • 对称：

    • nnn 为奇数：只能过一个端点对称，有 nnn 个置换，每种置换有 n+12\frac {n + 1}22n+1​ 个不同的置换环。

    • nnn 为偶数：可以过两个端点对称，也可以切两条边对称。

      • 过端点：有 n2\frac n22n​ 个置换，每种置换有 n2+1\frac n2 + 12n​+1 个置换环。

      • 过边：有 n2\frac n22n​ 个置换，每种置换 n2\frac n22n​ 个置换环。

置换群 GGG 的大小为 2n2n2n，所以把上面所有情况都加起来之后再除以 2n2n2n 就是答案。
