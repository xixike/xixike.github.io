# B. 【常中20180921】（第二套）C

解决这道题有两种方法：

  • 打表

  • 推结论

打表就不说了，这里来讲讲怎么推。

当 a=ba = ba=b 时，显然不合法。

假设 a>ba > ba>b，

那么 gcd⁡(a,b)≤a−b\gcd(a, b) \leq a - bgcd(a,b)≤a−b，a xor b≥a−ba\ xor\ b \geq a - ba xor b≥a−b。

很显然吧。

于是我们枚举一个 gcd⁡\gcdgcd 为 ddd，然后枚举它的倍数，判断 k×d xor (k−1)×dk \times d\ xor\ (k - 1) \times dk×d xor (k−1)×d 是否等于 ddd 即可。

# C. 【常中20180812T2】track

暴力能得 60pts 真 · 良心。

本题正解是 dpdpdp。

我们设 dp[i][j][k]dp[i][j][k]dp[i][j][k] 表示到了第 iii 个路口，匹配输入的字符串匹配到了第 jjj 位，此时的高度是 kkk 的方案数。

转移方程分两种情况：

  • 继续匹配下一个字符：dp[i+1][j+1][k+a[j+1]]+=dp[i][j][k]dp[i + 1][j + 1][k + a[j + 1]] += dp[i][j][k]dp[i+1][j+1][k+a[j+1]]+=dp[i][j][k]

  • 不匹配下一个字符：dp[i+1][Next(j)][k−a[j+1]]+=dp[i][j][k]dp[i + 1][Next(j)][k - a[j + 1]] += dp[i][j][k]dp[i+1][Next(j)][k−a[j+1]]+=dp[i][j][k]

Next(j)Next(j)Next(j) 就是先做个 kmpkmpkmp，然后在 nxtnxtnxt 数组上往前跳匹配点。

最后统计答案时，考虑正难则反。

先计算出总方案数，也就是卡特兰数 Cnn2−Cnn2−1C_n^{\frac n2} - C_n^{\frac n2 - 1}Cn2n​​−Cn2n​−1​。

然后减去不合法的即可。
