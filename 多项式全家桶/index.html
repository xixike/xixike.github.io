# 最后是多项式除法

除法由于其优秀的边界故不和上面放到一起，并且这里给出完整代码。

以及我这里是计算出 Q(x)Q(x)Q(x) 和 R(x)R(x)R(x) 后一块输出，由于计算 R(x)R(x)R(x) 时需要多项式卷积一下，因此对 Q(x)Q(x)Q(x) 做了一遍 NTTNTTNTT，所以卷积完之后一定记得要 NTTNTTNTT 回来啊。

来看看推导过程。

已知 F(x)F(x)F(x)，G(x)G(x)G(x)，求 F(x)=Q(x)G(x)+R(x)F(x) = Q(x)G(x) + R(x)F(x)=Q(x)G(x)+R(x)

FR(x)=an+an−1x+an−2x2+⋅⋅⋅+a0xnF_R(x) = a_n + a_{n - 1}x + a_{n - 2}x^2 +···+a_0x^nFR​(x)=an​+an−1​x+an−2​x2+⋅⋅⋅+a0​xn

容易发现，FR(x)=xnF(1x)F_R(x) = x^nF(\frac{1}{x})FR​(x)=xnF(x1​)

然后就可以愉快的推式子啦 :）

F(x)=Q(x)G(x)+R(x)F(1x)=Q(1x)G(1x)+R(1x)F(x) = Q(x)G(x) + R(x) \\ F(\frac{1}{x}) = Q(\frac{1}{x})G(\frac{1}{x}) + R(\frac{1}{x}) F(x)=Q(x)G(x)+R(x)F(x1​)=Q(x1​)G(x1​)+R(x1​)

F(x)F(x)F(x) 是 nnn 次的，G(x)G(x)G(x) 是 mmm 次的，Q(x)Q(x)Q(x) 是 n−mn - mn−m 次的，R(x)R(x)R(x) 是 n−m−1n - m - 1n−m−1 次的，所以两边同乘 xnx^nxn：

xnF(x)=xn−mQ(x)∗xmG(x)+xn−m−1R(x)∗xm+1xnF(x)≡xn−mQ(x)∗xmG(x)  (mod  xn−m−1)FR(x)≡QR(x)∗GR(x)  (mod  xn−m−1)QR(x)≡FR(x)∗GR(x)−1  (mod  xn−m−1)x^nF(x) = x^{n - m}Q(x) * x^{m}G(x) + x^{n - m - 1}R(x) * x^{m + 1} \\ x^nF(x) \equiv x^{n - m}Q(x) * x^{m}G(x)\ \ (mod\ \ x^{n - m - 1}) \\ F_R(x) \equiv Q_R(x) * G_R(x)\ \ (mod\ \ x^{n - m - 1}) \\ Q_R(x) \equiv F_R(x) * G_R(x)^{-1}\ \ (mod\ \ x^{n - m - 1}) xnF(x)=xn−mQ(x)∗xmG(x)+xn−m−1R(x)∗xm+1xnF(x)≡xn−mQ(x)∗xmG(x)  (mod  xn−m−1)FR​(x)≡QR​(x)∗GR​(x)  (mod  xn−m−1)QR​(x)≡FR​(x)∗GR​(x)−1  (mod  xn−m−1)

FR(x)F_R(x)FR​(x) 和 GR(x)G_R(x)GR​(x) 都是已知的（预处理一下即可），Q(x)Q(x)Q(x) 就是 QR(x)Q_R(x)QR​(x) 的系数倒过来。
