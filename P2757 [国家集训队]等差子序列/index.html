# Solution

我们直接找是否有等于 3 的等差子序列即可。

假设三个数 ai<aj<aka_i < a_j < a_kai​<aj​<ak​ 形成了等差子序列，那么有 ai+ak=2×aja_i + a_k = 2 \times a_jai​+ak​=2×aj​，所以对于一个 aia_iai​ 对应的 aka_kak​ 只有一个。

因此我们对于每一个 aja_jaj​ 只需要判断一对 ai,aka_i, a_kai​,ak​ 是否在 jjj 的两边。

如何判断呢？很简单。

我们开一个权值线段树，依次往里面插入数，把相应的位置赋值为 1。

假设已经枚举到了第 jjj 个数，此时 a1∼aj−1a_1 \sim a_{j - 1}a1​∼aj−1​ 都已经在权值线段树里面了。

我们只需要判断 tj−len∼jt_{j - len \sim j}tj−len∼j​ 和 tj∼j+lent_{j \sim j + len}tj∼j+len​ 的 hash 值一不一样即可（ttt 为权值线段树，len=min⁡(aj−1,n−aj)len = \min(a_j - 1, n - a_j)len=min(aj​−1,n−aj​) 防止越界）。

思考一下这样为什么是正确的。

假设 hash 值一样，表示所有能与 aja_jaj​ 配对的 ai,aka_i, a_kai​,ak​ 都已经在 a1∼aj−1a_1 \sim a_{j - 1}a1​∼aj−1​ 出现过了，自然不可能组成等差子序列。

反之自然可以。
