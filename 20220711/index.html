# A. 饥饿的小鸟（river）

对于一个位置 iii，枚举 i−Li - Li−L 到 i−1i - 1i−1，显然从越早的荷叶跳过来越优，开个指针记录一下枚举到的位置即可。

复杂度 O(n)O(n)O(n)

# B. 进化序列（evolve）

非常水的题。

开个桶维护二进制下每一位出现次数，显然对于 lll 单调向右，最靠右合法的 rrr 是单调递增的。

所以像莫队一样维护桶即可，复杂度 O(nlog⁡V)O(n \log V)O(nlogV)。

# C. 旅行 travel

首先 E(x2)≠E(x)×E(x)E(x^2) \neq E(x) \times E(x)E(x2)=E(x)×E(x)，因为事件不独立。

但是有 E(x+x)=E(x)+E(x)E(x + x) = E(x) + E(x)E(x+x)=E(x)+E(x)

所以推一下：E((a+b)2)=E(a2+b2+2ab)=E(a2)+E(b2)+E(2ab)E((a + b)^2) = E(a^2 + b^2 + 2ab) = E(a^2) + E(b^2) + E(2ab)E((a+b)2)=E(a2+b2+2ab)=E(a2)+E(b2)+E(2ab)

以下两个数组都是以 1 为根的情况下定义的。

我们设 hxh_xhx​ 表示 xxx 子树内的点的 FFF 和的期望，fxf_xfx​ 表示子树内 FyF_yFy​ 和的平方的期望。

显然有：

hx=(1−p)×hx+p×(hx+hy)fx=(1−p)×fx+p×(fx+fy+2hxhy)h_x = (1 - p) \times h_x + p \times (h_x + h_y) \\ f_x = (1 - p) \times f_x + p \times (f_x + f_y + 2h_xh_y) hx​=(1−p)×hx​+p×(hx​+hy​)fx​=(1−p)×fx​+p×(fx​+fy​+2hx​hy​)

（注意要先转移 fff，再转移 hhh）

设 gxg_xgx​ 为以 1 为根的情况下 xxx 子树外（不包括 xxx）的 FFF 的和的期望，ansxans_xansx​ 为答案，即 xxx 连通块的 FFF 的和的平方的期望。

接下来看如何换根。

一个比较无脑的思路是维护 xxx 儿子的前缀后缀和，然后各种转移，这里给一个代码简洁很多的做法（当然要推式子）。

考虑根从 uuu 换到 vvv。

gv=(1−p)×hv+p×(hv+E1)g_v = (1 - p) \times h_v + p \times (h_v + E_1) gv​=(1−p)×hv​+p×(hv​+E1​)

E1E_1E1​ 为当前情况下 vvv 以外的点 FFF 的和的期望。那么 E1E_1E1​ 怎么求呢？

观察到对于 gug_ugu​ 我们有这样的式子：

gu=(1−p)×E1+p×(hv+E1)g_u = (1 - p) \times E_1 + p \times (h_v + E_1) gu​=(1−p)×E1​+p×(hv​+E1​)

而这个时候 gug_ugu​ 是已知的，所以直接计算出 E1E_1E1​ 再代回到刚才的式子中计算 gvg_vgv​ 即可。

ansansans 数组也是同理的，对于 ansvans_vansv​ 有：

ansv=(1−p)×fv+p×(fv+E2+2hvE1)ans_v = (1 - p) \times f_v + p \times (f_v + E_2 + 2h_vE_1) ansv​=(1−p)×fv​+p×(fv​+E2​+2hv​E1​)

这就相当于 E((a+b)2)=E(a2)+E(b2)+E(2ab)E((a + b)^2) = E(a^2) + E(b^2) + E(2ab)E((a+b)2)=E(a2)+E(b2)+E(2ab)

同理，ansuans_uansu​ 有：

ansu=(1−p)×E2+p×(fv+E2+2hvE1)ans_u = (1 - p) \times E_2 + p \times (f_v + E_2 + 2h_vE_1) ansu​=(1−p)×E2​+p×(fv​+E2​+2hv​E1​)

也是计算出 E2E_2E2​ 再代回去算 ansvans_vansv​ 即可。

代码还是有一定细节的，最重要的是不要忘记取模。

# D. 平衡树 splay

考场上一眼是个 LCT\text{LCT}LCT 裸题，但是没想到 rotaterotaterotate 之后中序遍历不变实在是降智了。

不难发现，旋转之后树的中序遍历不变。

所以我们对中序遍历维护一个线段树，每次旋转就相当于单点修改了两个数的值。

然后维护区间乘积即可。

代码也是十分的板子，rotaterotaterotate 不要写挂了就好/kk
