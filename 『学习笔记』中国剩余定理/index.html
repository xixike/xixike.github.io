# 中国剩余定理 (CRT)

中国剩余定理是用来解决如下问题的：

{x≡a1(modm1)x≡a2(modm2)...x≡an(modmn)\begin{cases} x\equiv a_1\ \pmod{m_1}&\\ x\equiv a_2\ \pmod{m_2}&\\ ...\ &\\ x\equiv a_n\ \pmod{m_n} \end{cases}⎩⎪⎪⎪⎪⎨⎪⎪⎪⎪⎧​x≡a1​ (modm1​)x≡a2​ (modm2​)... x≡an​ (modmn​)​​

注意：模数必须互质。

# 解法

我们设

M=∏i=1nmiMi=MmiMiti≡1(modmi)M = \prod_{i = 1}^nm_i\\ M_i = \frac{M}{m_i}\\ M_it_i\equiv1\pmod{m_i}M=i=1∏n​mi​Mi​=mi​M​Mi​ti​≡1(modmi​)

tit_iti​ 相当于是 MiM_iMi​ 在模 mim_imi​ 意义下的逆元。

此时我们可以构造出一组解：

x=∑i=1naiMitix = \sum_{i = 1}^na_iM_it_ix=i=1∑n​ai​Mi​ti​

进而推出任意解：

x0=x+k×Mx_0 = x + k \times Mx0​=x+k×M

# 证明

根据数组的含义，容易得到，对于第 iii 个同余方程，当 j≠ij \ne ij=i 时，都有：

ajMjtj≡0(modmi)a_jM_jt_j \equiv 0 \pmod {m_i}aj​Mj​tj​≡0(modmi​)

因为 MjM_jMj​ 中包含 mim_imi​。

当 j=ij = ij=i 时，有：

∑j=1najMjtj≡ai(modmi)\sum_{j = 1}^na_jM_jt_j \equiv a_i \pmod {m_i}j=1∑n​aj​Mj​tj​≡ai​(modmi​)

证毕.

逆元需要用到扩展欧几里得算法 (exgcd) 来求解，这里不再赘述，不会的左转百度一下吧 QwQ

# 扩展中国剩余定理 (exCRT)

注意到多了扩展，那么这个扩展在哪里呢？

普通的 CRT 要求模数必须互质，这就很恶心，exCRT 就解决了这个问题，也就是模数不互质时的解决方法。

# 解法

同样是这个式子：

{x≡a1(modm1)x≡a2(modm2)...x≡an(modmn)\begin{cases} x\equiv a_1\ \pmod{m_1}&\\ x\equiv a_2\ \pmod{m_2}&\\ ...\ &\\ x\equiv a_n\ \pmod{m_n} \end{cases}⎩⎪⎪⎪⎪⎨⎪⎪⎪⎪⎧​x≡a1​ (modm1​)x≡a2​ (modm2​)... x≡an​ (modmn​)​​

模数不互质时怎么办呢？

我们考虑能否把两个同余方程合并为一个。

我们对两个方程进行讨论：

{x≡a1(modm1)x≡a2(modm2)\begin{cases} x\equiv a_1\ \pmod{m_1}&\\ x\equiv a_2\ \pmod{m_2}& \end{cases}{x≡a1​ (modm1​)x≡a2​ (modm2​)​​

同余也可以表示为：

x=k1m1+a1=k2m2+a2x = k_1m_1 + a_1 = k_2m_2 + a_2x=k1​m1​+a1​=k2​m2​+a2​

移项：

k1m1−k2m2=a2−a1k_1m_1 - k_2m_2 = a_2 - a_1k1​m1​−k2​m2​=a2​−a1​

诶，这不就是一个 exgcd 的形式吗，所以容易判断是否有解：

  • gcd⁡(m1,m2)∣(a2−a1)\gcd(m_1, m_2) \mid (a_2 - a_1)gcd(m1​,m2​)∣(a2​−a1​)，有解。
  • 否则无解。

继续来看这个式子，设：

d=gcd⁡(m1,m2)p1=m1dp2=m2dd = \gcd(m_1, m_2)\\ p_1 = \frac{m_1}{d}\\ p_2 = \frac{m_2}{d}d=gcd(m1​,m2​)p1​=dm1​​p2​=dm2​​

显然 p1p_1p1​ 和 p2p_2p2​ 是互质的，代入到原式中：

k1p1−k2p2=a2−a1dk_1p_1 - k_2p_2 = \frac{a_2 - a_1}{d}k1​p1​−k2​p2​=da2​−a1​​

这个式子中，左边满足了 gcd⁡(p1,p2)=1\gcd(p1, p2) = 1gcd(p1,p2)=1，右边当且仅当 d∣(a2−a1)d \mid (a_2 - a_1)d∣(a2​−a1​) 才有解，利用 exgcd 求出一组特解：

x1p1+x2p2=1x_1p_1 + x_2p_2 = 1x1​p1​+x2​p2​=1

然后可以直接算出 k1,k2k_1,k_2k1​,k2​：

{k1=a2−a1dx1k2=−a2−a1dx2\begin{cases} k1 = \frac{a_2 - a_1}{d}x_1&\\ k2 = -\frac{a_2 - a_1}{d}x_2& \end{cases}{k1=da2​−a1​​x1​k2=−da2​−a1​​x2​​​

于是：

x=k1m1+a1=a2−a1dx1mi+a1x = k_1m_1 + a_1 = \frac{a_2 - a_1}{d}x_1m_i + a_1x=k1​m1​+a1​=da2​−a1​​x1​mi​+a1​

至此，我们成功构造出了一个解，但是整个解系如何构造呢？

定理： 我们有一个特解 xxx，通解为:

x0≡x(modlcm(m1,m2))x_0 \equiv x \pmod {lcm(m_1, m_2)}x0​≡x(modlcm(m1​,m2​))

且 0,1,2⋅⋅⋅lcm(m1,m2)0, 1, 2 ···lcm(m_1, m_2)0,1,2⋅⋅⋅lcm(m1​,m2​) 中只有一个解。
