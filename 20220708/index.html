# C. 十一 eleven

next_permutation 谁不会啊，喜提 20 分。

观察到部分分：每个数字位数相同。

考场上实在是想不出来这玩意有啥用 /kk

稍微模拟一下可知：

  • 对于偶数位数的数：数字 xxx 经过这个数之后形成的数一定是 (x+?)mod11(x + ?) \bmod 11(x+?)mod11 这样形式的。

  • 对于奇数位数的数：数字 xxx 经过之后一定是 (x−?)mod11(x - ?) \bmod 11(x−?)mod11 这样子的。

先考虑部分分。

如果所有数都是偶数位数的，那么最终的数为：

(((0+a)mod11+b)mod11)⋯(((0 + a) \bmod 11 + b) \bmod 11)\cdots(((0+a)mod11+b)mod11)⋯

所以我们只需要确定一下所有数的和是否能整除 11 即可，答案就是 n!n!n!。

对于奇数位数，表达式为：

(c−(b−(0+a)))(c - (b - (0 + a)))(c−(b−(0+a)))

展开之后为：

−0+a−b+c- 0 + a - b + c−0+a−b+c

不难发现这东西正负号是交替的，所以设 fi,j,kf_{i, j, k}fi,j,k​ 表示 iii 个正，jjj 个负，这些数的和 mod11\bmod 11mod11 的值为 kkk 时的方案数。

转移的时候对于新加进来的数放到正数位置还是负数位置分别考虑：

有转移方程：

fi+1,j,k+nxtp+=fi,j,kfi,j+1,k−nxtp+=fi,j,kf_{i + 1, j, k + nxt_p} += f_{i, j, k} \\ f_{i, j + 1, k - nxt_p} += f_{i, j, k}fi+1,j,k+nxtp​​+=fi,j,k​fi,j+1,k−nxtp​​+=fi,j,k​

但是一个奇数位数的数展开之后的正负号个数是固定的，分别为 cntz,cntfcnt_z, cnt_fcntz​,cntf​（自己推一下吧）

部分分就做完了。

再来看满分如何做。

我们对偶数位数的数也进行一次上述 dp\text{dp}dp，设为 gi,j,kg_{i, j, k}gi,j,k​。然后思考如何将两个 dp\text{dp}dp 数组合并。

我们考虑把偶数位数的数插入到奇数位数的数中间，所以有些偶数位数的数对答案的贡献是正，有些是负的。

枚举有多少个是正的，多少个是负的。

dp\text{dp}dp 值贡献为 fcntz,cntf,11−k×gi,cnt−i,kf_{cnt_z, cnt_f, 11 - k} \times g_{i, cnt - i, k}fcntz​,cntf​,11−k​×gi,cnt−i,k​，但是偶数位数的数还有很多种放法。

  • 对于奇数位数中的正数：

    有 cntz!cnt_z!cntz​! 种排列方式，然后新插入一个数，有 cntzcnt_zcntz​ 个位置，对于插入的第二个数，有 cntz+1cnt_z + 1cntz​+1 个位置，以此类推。

    所以总方案数为 (cntz+i−1)!×cntz(cnt_z + i - 1)! \times cnt_z(cntz​+i−1)!×cntz​

  • 对于计数位数中的负数：

    实际上是有 cntf+1cnt_f + 1cntf​+1 个位置的，因为还要加上 −0-0−0 的位置，例子见上面。所以放第一个数时就有 cntf+1cnt_f + 1cntf​+1 个位置，第二个数有 cntf+2cnt_f + 2cntf​+2 个位置，以此类推。

    总方案数为 (cntf+cnt−i)!(cnt_f + cnt - i)!(cntf​+cnt−i)!

# D. 棋盘 chess

一道区间 dp\text{dp}dp 板题，然而考场上并没有想到 /kk

首先曼哈顿距离转切比雪夫距离，坐标从 (x,y)(x, y)(x,y) 变成了 (x−y,x+y)(x - y, x + y)(x−y,x+y)。

然后所求距离从 ∣x1−x2∣+∣y1−y2∣|x_1 - x_2| + |y_1 - y_2|∣x1​−x2​∣+∣y1​−y2​∣ 变成了 max⁡(x1−x2,y1−y2)\max(x_1-x_2,y_1-y_2)max(x1​−x2​,y1​−y2​)

设 fl,r,d,uf_{l, r, d, u}fl,r,d,u​ 表示还剩这个矩形时，消掉这个矩形内所有点的最小代价。

使用记忆化搜索实现，每次转移的时候从四条边分别往里收缩一个单位即可。
