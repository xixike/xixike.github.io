# A. 「2017 山东二轮集训 Day1」第二题

看起来是这套题里面最简单的了。

大力贪心。

每次找到最低点，然后分治处理两边。

答案分别加上两边的大小除以 kkk，然后多余的部分送给最低点统计。

# B. 「2017 山东二轮集训 Day1」第一题

毒瘤数据结构。

大致思路是找出每个点向右能到达的最远的点，然后把这个贡献加到主席树里，查询的时候直接在主席树上查询 l∼rl \sim rl∼r 的区间和即可。

那么如何预处理呢？

设 s[0/1][i][j]s[0/1][i][j]s[0/1][i][j] 为前 iii 个数的异或和，最高位 jjj 从 1→01 \rightarrow 01→0 和 0→10 \rightarrow 10→1 的个数，sumisum_isumi​ 为前缀异或和。

对于区间 l∼rl \sim rl∼r，如果 suml−1sum_{l - 1}suml−1​ 的最高位 jjj 为 1，那么 l∼rl \sim rl∼r 中第 jjj 位就不能出现 1→01 \rightarrow 01→0，反之不能出现 0→10 \rightarrow 10→1。

所以可以使用二分查找。

然后在主席树进行区间加，这需要使用标记永久化。

所谓标记永久化其实就是修改时不写 pushdown\text{pushdown}pushdown，只在查询的时候加上标记的贡献即可。

# C. GCD

这道题就相当之困难了。

题目让我们求 gcd⁡(afn+bfn+1,cfn+dfn+1)\gcd(af_n + bf_{n + 1}, cf_n + df_{n + 1})gcd(afn​+bfn+1​,cfn​+dfn+1​)。

首先通过更相减损术可以去掉 dfn+1df_{n + 1}dfn+1​ 这一项。

式子就变成了 gcd⁡(a′fn+b′fn+1,c′fn)\gcd(a'f_n + b'f_{n + 1}, c'f_n)gcd(a′fn​+b′fn+1​,c′fn​)。

我们把 c′c'c′ 和 fnf_nfn​ 分开计算：

  • gcd⁡(afn+bfn+1,fn)=gcd⁡(bfn+1,fn)\gcd(af_n + bf_{n + 1}, f_n) = \gcd(bf_{n + 1}, f_n)gcd(afn​+bfn+1​,fn​)=gcd(bfn+1​,fn​)。

    通过打表发现，gcd⁡(fn,fn+1)=3n2\gcd(f_n, f_{n + 1}) = 3^{\frac n2}gcd(fn​,fn+1​)=32n​，所以就是 gcd⁡(b,fn)×3n2\gcd(b, f_n) \times 3^{\frac n2}gcd(b,fn​)×32n​。

  • gcd⁡(afn+bfn+1,c)\gcd(af_n + bf_{n + 1}, c)gcd(afn​+bfn+1​,c)，设 g=gcd⁡(afn+bfn+1,fn)g = \gcd(af_n + bf_{n + 1}, f_n)g=gcd(afn​+bfn+1​,fn​)，然后先除掉这一部分，即求 gcd⁡(afn+bfn+1g,c)\gcd(\frac {af_n + bf_{n + 1}}g, c)gcd(gafn​+bfn+1​​,c)，然后让左边模 gcgcgc 计算即可。

所以答案就是：
