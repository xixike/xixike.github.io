# A. 核酸检测

看了很久才发现是斜率优化……那么就是一个斜率优化板题。

先看暴力 dp\text{dp}dp，设 fif_ifi​ 表示计算到 iii 位置且在 iii 上放一名医护人员的最小花费，转移的时候枚举上一名医护人员的位置。

fi=min⁡{fj+(i−j)×(i−j−1)2+ai}f_i = \min\{f_j + \frac {(i - j) \times (i - j - 1)} 2 + a_i\} fi​=min{fj​+2(i−j)×(i−j−1)​+ai​}

化简一下就可以斜率优化了（自己推吧，懒得打了QwQ）。

# B. 排列

这题也比较水。

开个值域线段树维护数字是否出现，这是个非常经典的东西，就维护 maxl,maxr,maxzdmaxl, maxr, maxzdmaxl,maxr,maxzd 三个值就好。

外面再套一个莫队就可以做到 O(nnlog⁡n)O(n \sqrt n \log n)O(nn​logn)，不是很卡。

是在不行的话莫队排序的时候加个奇偶优化即可。

# C. ROT-Tree Rotations

建权值线段树，然后线段树合并。

合并时计算左右子树在前在后时分别的逆序对数，然后取较小的即可。

# D. 计算

数位 dp\text{dp}dp。

先通过 kmp\text{kmp}kmp 预处理出 mmm 的 nxtnxtnxt 数组，方便后面转移。

设 fi,jf_{i, j}fi,j​ 表示计算到 nnn 的第 iii 位，匹配到 mmm 的第 jjj 位时后面的答案。

转移时枚举第 iii 位为 xxx，那么贡献为：fi+1,j′×ex×10tnf_{i + 1, j'} \times e^{\frac{x \times 10^t}n}fi+1,j′​×enx×10t​。

最后输出时要减去 e0e^0e0，即减 1。
