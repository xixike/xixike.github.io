# P2513 [HAOI2009] 逆序对数列

# Description

Luogu 传送门

# Solution

看到名字，本来以为又是树状数组维护什么东西，结果不是……

看了一眼题解发现要 dpdpdp 一下子，dpdpdp 就 dpdpdp 吧。

我们设 fi,jf_{i,j}fi,j​ 表示 iii 的全排列中，逆序对数为 jjj 的个数。

我们考虑把 iii 插入到 i−1i - 1i−1 的全排列中，枚举这次插入增加了 kkk 个逆序对。

那么转移方程就是：

fi,j=∑k=max{0,j−i+1}jfi−1,kf_{i, j} = \sum\limits_{k = max\{0, j - i + 1\}}^j{f_{i - 1, k}}fi,j​=k=max{0,j−i+1}∑j​fi−1,k​

朴素的转移是 O(n2)O(n^2)O(n2) 的，考虑优化。

优化也很简单，发现 kkk 是从 0 开始枚举的，做个前缀和即可。

# P2607 [ZJOI2008] 骑士

Luogu 传送门

# Solution

基环树上 dpdpdp 板子。

（但是为什么全机房除了我其他人都做过啊啊啊，果然还是我太菜了 QwQQwQQwQ）

这道题给出的骑士之间的厌恶关系会形成基环树（森林），那么我们要在上面跑 dpdpdp。

以下按一棵基环树来讨论。

我们先找到环，然后选择上面的一个点为根，跑树形 dpdpdp，然后再换成与它相邻的一个点为根跑一遍树形 dpdpdp，取个最大值就是答案了（树形 dpdpdp 最下面再说）。

我们再来考虑基环树森林，其实是一样的。

主函数里枚举一遍所有的点，如果还没遍历过，就重复一次上述操作。

# P4035 [JSOI2008] 球形空间产生器

# Description

Luogu 传送门

# Solution

高斯消元板子

不过还是要稍稍推一下式子的。

关于 nnn 维空间里两个点之间的距离题目下方的说明 / 提示里有。

这道题给我们的都是表面上的点，言外之意，到球心距离相等。

假设我们要求的球心坐标：(x1,x2,x3……xn)(x_1, x_2, x_3……x_n)(x1​,x2​,x3​……xn​)

那么有：∑i=1n∑j=1n(ai,j−xj)2=C\sum\limits_{i = 1}^{n}{\sum\limits_{j = 1}^n(a_{i,j} - x_j)^2 = C}i=1∑n​j=1∑n​(ai,j​−xj​)2=C

但是这是个二次方程，还有个常数怎么办啊？

上下一减就完了，减的过程就不写了（懒得打字了），建议自己手推一下。

下面是化简完的式子：

∑j=1n2(ai,j−ai+1,j)xj=∑j=1n(ai,j2−ai+1,j2)\sum\limits_{j = 1}^n2(a_{i,j} - a_{i + 1, j})x_j = \sum\limits_{j = 1}^n(a_{i, j}^2 - a_{i + 1, j}^2)j=1∑n​2(ai,j​−ai+1,j​)xj​=j=1∑n​(ai,j2​−ai+1,j2​)

看着不清新？换元一下：

k=2(ai,j−ai+1,j)k = 2(a_{i, j} - a_{i + 1, j})k=2(ai,j​−ai+1,j​)

y=∑j=1n(ai,j2−ai+1,j2)y = \sum\limits_{j = 1}^n(a_{i, j}^2 - a_{i + 1, j}^2)y=j=1∑n​(ai,j2​−ai+1,j2​)

∑j=1nkj×xj=yj\sum\limits_{j = 1}^nk_j \times x_j = y_jj=1∑n​kj​×xj​=yj​

对这个式子建个增广矩阵，跑个高斯消元就完了。
