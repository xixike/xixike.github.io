# CF662C Binary Table

非常巧妙的题。

首先观察到 n≤20n \leq 20n≤20 但是 m≤105m \leq 10^5m≤105 是什么鬼？

先想一个比较显然的暴力，我们 2n2^n2n 枚举每一行是否翻转，然后每一列的状态就都是已知的了，O(m)O(m)O(m) 枚举所有的列，答案加上当前列 0/10/10/1 出现次数较小值即可，时间复杂度 O(m2n)O(m2^n)O(m2n)。

那么显然这个 mmm 实在是太大了，考虑如何去掉 mmm。

一共有 mmm 种列状态，每种状态都是 0∼2n−10 \sim 2^n - 10∼2n−1 里的一种，我们记 aia_iai​ 表示状态 iii 出现的次数，bib_ibi​ 表示状态 iii 中 0/10/10/1 出现次数的较小值。

假设当前枚举的行状态为 stastasta，那么答案就是：

ans=∑i=02n−1ai×bi⊕staans = \sum_{i = 0}^{2^n - 1} a_i \times b_{i \oplus sta}ans=i=0∑2n−1​ai​×bi⊕sta​

状态为 iii 的有 aia_iai​ 列，翻转之后每一列上 1 的个数都为 bi⊕stab_{i \oplus sta}bi⊕sta​。

设 fif_ifi​ 表示状态为 iii 时的答案，那么有：

fk=∑i⊕j=kai×bjf_k = \sum_{i \oplus j = k} a_i \times b_jfk​=i⊕j=k∑​ai​×bj​

所以求一下 aia_iai​ 和 bib_ibi​ 的异或卷积，然后统计答案即可。

# CF850E Random Elections

这是我见过的最巧妙的 FWTFWTFWT 题目。

题意就不解释了。

显然一轮中赢两次的玩家之后一个，所以计算一个人获胜的次数乘 3 即可。

考虑一个人如何才能获胜。

对于两场比赛 A−B,C−AA - B, C - AA−B,C−A（题目要求），AAA 的的票状态分别为 S1,S2S_1, S_2S1​,S2​，假设它们第 iii 位的情况为 (x,y)(x, y)(x,y)（ x,y=0/1x, y = 0/1x,y=0/1），那么有 4 中情况：

  • (x,y)=(0,0)⇒CAB,CBA(x, y) = (0, 0) \Rightarrow CAB, CBA(x,y)=(0,0)⇒CAB,CBA
  • (x,y)=(1,0)⇒CAB(x, y) = (1, 0) \Rightarrow CAB(x,y)=(1,0)⇒CAB
  • (x,y)=(0,1)⇒BAC(x, y) = (0, 1) \Rightarrow BAC(x,y)=(0,1)⇒BAC
  • (x,y)=(1,1)⇒ABC,ACB(x, y) = (1, 1) \Rightarrow ABC, ACB(x,y)=(1,1)⇒ABC,ACB

后面的字母串表示这个人可能的投票顺序。

根据乘法原理，(1,0)(1, 0)(1,0) 和 (0,1)(0, 1)(0,1) 都是没有贡献的（均为乘 1），(0,0)(0, 0)(0,0) 和 (1,1)(1, 1)(1,1) 均有 2 的贡献。

所以设 S=S1⊕S2S = S_1 \oplus S_2S=S1​⊕S2​，cntcntcnt 为 SSS 中 0 的个数，那么方案数就是 2cnt2^{cnt}2cnt。

给 fsf_sfs​ 赋好初值以后，不难发现这东西就是一个 fff 自己和自己的异或卷积！

于是这题就愉快的解决了。
