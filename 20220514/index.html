# A. 越野赛车问题

考虑对值域进行线段树分治，把在分治区间 [L,R][L, R][L,R] 的边连上，动态维护树的直径即可。

关于如何维护直径，就是把原来直径的两个端点和新加进来的边的两个端点组合一下，距离最远的两个点就是新的直径。

再开个可撤销并查集维护历史直径信息，就可以动态更新了。

直接贺板子，好耶。

# C. 椅子

AT2645 [ARC076D] Exhausted?

可以直接暴力最大流，能拿不少部分分。

然后考场上以为是线段树优化建图什么的，然而并不是。

本题大力贪心即可。

首先一个假贪心：

  • 按照 lil_ili​ 从小到大排序，相同的按 rir_iri​ 从小到大排序。

  • 对于每个人先在左边找，让他在 ≤li\leq l_i≤li​ 的最右边的空位置坐下，如果左边没有位置了，让他去右边找，在 ≥ri\geq r_i≥ri​ 的最左边的空位置坐下，如果都没有就加一张椅子。

如何避免这一情况呢？

重新考虑这一过程，还是和上面一样的操作。

我们考虑到了第 iii 个人，在 lil_ili​ 左边找到了空位置，让 iii 坐下。

这时第 jjj 个人来了，但是我们在 ljl_jlj​ 左边找不到空位置了，我们考虑是否让 iii 起来，去右边找位置，并让 jjj 坐在 iii 坐的位置上（排过序了，所以 iii 坐的位置 jjj 肯定能坐）。

如何维护这一“反悔”操作呢？

我们开一个小根堆，如果 iii 在 ≤li\leq l_i≤li​ 找到了空位置，我们把 rjr_jrj​ 插入到小根堆中。

还是考虑如果 iii 在左边找不到位置，此时堆顶为 rjr_jrj​，我们分两种情况讨论：

  • ri<rjr_i < r_jri​<rj​，那就直接让 iii 去右边找个空位置坐下就行。
  • ri≥rjr_i \geq r_jri​≥rj​，我们让 iii 去坐 jjj 的位置，然后让 jjj 去 ≥rj\geq r_j≥rj​ 找空位置。

然后就 ok\text{ok}ok 了。

# D. 方舟系统

本来是没这题的，但是 B\text{B}B 题大家之前都做过，所以就加了道题 /kk

O(n2)O(n^2)O(n2) 的暴力 dp\text{dp}dp 非常好想，设 fif_ifi​ 表示处理到了第 iii 个数，最少的代价。

转移方程：

fi=min⁡{fj−1+∣aj−(i−j)∣}f_i = \min\{f_{j- 1} + |a_j - (i - j)| \} fi​=min{fj−1​+∣aj​−(i−j)∣}

能得 60pts\text{60pts}60pts，好耶。

那么如何优化呢？

我们转移过程中就取了个最小值，所以我们只要能够维护一手前缀最小值即可。

先把这个绝对值打开，即 aj−(i−j)a_j - (i - j)aj​−(i−j) 或者 (i−j)−aj(i - j) - a_j(i−j)−aj​。

iii 是定值，不用管它，所以就剩下了 aj+ja_j + jaj​+j 和 −aj−j-a_j - j−aj​−j。

所以我们开两个树状数组，分别维护 fj−1+aj+jf_{j - 1} + a_j + jfj−1​+aj​+j 和 fj−1−aj−jf_{j - 1} - a_j - jfj−1​−aj​−j 的最小值。

不要忘记这两个值可不是想用就能用的，绝对值的符号是固定的，所以一个是前缀最小值，一个是后缀最小值，自己判断一下吧。
