# A. Everybody Likes Good Arrays!

水题。

奇乘奇，偶乘偶的奇偶性都不变。数一下有多少段即可。

# B. Emordnilap

看到题目让求 1∼n1 \sim n1∼n 的所有排列的 S(p)S(p)S(p) 的和，而且这只是道 B\text{B}B。

大胆猜测所有排列的 S(p)S(p)S(p) 是有一定关系的，再自己推一推，模拟一下。

不难发现 S(p)S(p)S(p) 就是个定值（但是我忘了是多少了qwq）.

# C. Quiz Master

看到极差最小，自然是要二分了。

现预处理出来所有 aia_iai​ 包含的小于等于 mmm 的因数有哪些。

二分判断的时候双指针扫，开个桶记录当前出现了多少个 mmm 以内的不同的因数，如果等于 mmm 就合法。

# D. Score of a Tree

感觉非常妙的一道题。

首先不难发现，节点 uuu 在 ttt 时刻的值为它的子树内所有深度为 depu+tdep_u + tdepu​+t 的点的异或和。

假设 uuu 子树内深度为 depu+kdep_u + kdepu​+k 的点有 cntcntcnt 个。

我们考虑什么样的权值分配对 uuu 是有贡献的，当且仅当异或和为 1，即有奇数个 1。

根据二项式基本功得出也就是 2cnt−12^{cnt - 1}2cnt−1。此时其他的点都可以随便选，也就是 2n−cnt2^{n - cnt}2n−cnt.

乘到一起就是 2n−12^{n - 1}2n−1。

dfs\text{dfs}dfs 过程中对每个点都算一下答案即可。

# E. Edge Reverse

感觉比上面那道题简单不少。

看到最大权值最小肯定是要二分的。

现在的问题就是如何快速 check\text{check}check。

其实也很简单，我们把 w≤midw \leq midw≤mid 的边全都变成无向边，然后跑 tarjantarjantarjan 缩个点看一看是不是只有一个入度为 0 的点即可。

# F. Comfortably Numb

肉眼可见的板子题。

al⊕al+1⊕⋯⊕ar−1⊕ara_l \oplus a_{l + 1} \oplus \cdots \oplus a_{r - 1} \oplus a_{r}al​⊕al+1​⊕⋯⊕ar−1​⊕ar​：明显就是个可持久化 01trie\text{01trie}01trie 板子啊。

max⁡{a[l⋯r]}\max\{a[l \cdots r] \}max{a[l⋯r]}：明显就是笛卡尔树板子啊。

所以正确的做法就是先建出笛卡尔树，然后在 xxx 较小的儿子里枚举一个边界，在另一个儿子里查询需要的异或最大值。

复杂度 2 只 log⁡\loglog。
