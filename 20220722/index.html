# A. 数正方体

推了 2.5h 没有推出来，心态非常崩。

主要是没有网通项式方面去想，考场上推出来递推式之后一直在优化递推……

通过手模不难发现，fi,j=fi−1,j−1+2×fi−1,jf_{i, j} = f_{i - 1, j - 1} + 2 \times f_{i - 1, j}fi,j​=fi−1,j−1​+2×fi−1,j​.

然后再打一个更大的表，不难发现

fn,m=2n−m(nm)f_{n, m} = 2^{n - m} \dbinom {n}{m} fn,m​=2n−m(mn​)

然后 O(n)O(n)O(n) 预处理，O(1)O(1)O(1) 查询就好了。

# B. 数连通块

首先把题目中的树跑出来，设 pip_ipi​ 表示 iii 点存在的概率，qiq_iqi​ 表示 iii 点不存在的概率。

对于询问 l∼rl \sim rl∼r，考虑每个点对连通块个数的贡献。

设当前点为 xxx，分类讨论：

  • faxfa_xfax​ 在 l∼rl \sim rl∼r 中，xxx 想要对答案有贡献，那么必须 faxfa_xfax​ 不存在，且 xxx 存在，所以贡献为 q[fax]×p[x]q[fa_x] \times p[x]q[fax​]×p[x]。

  • faxfa_xfax​ 不在 l∼rl \sim rl∼r 中，只要 xxx 存在就对答案就贡献，所以贡献为 p[x]p[x]p[x]。

暴力是 O(n2)O(n^2)O(n2) 的，期望得分 10pts\text{10pts}10pts，考虑使用莫队优化，然后就可以过了……
