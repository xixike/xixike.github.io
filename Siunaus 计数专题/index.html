# 二项式反演

# 公式

fn=∑i=0n(ni)gi⇔gn=∑i=0n(−1)n−i(ni)fif_n = \sum_{i = 0}^n \dbinom {n}{i}g_i \Leftrightarrow g_n = \sum_{i = 0}^n (-1)^{n - i} \dbinom ni f_ifn​=i=0∑n​(in​)gi​⇔gn​=i=0∑n​(−1)n−i(in​)fi​

证明就不细说了。那么二项式反演的式子有什么用呢？

恰好和至多的转换：

设 fkf_kfk​ 为恰好 kkk 个的方案数，gkg_kgk​ 为至多 kkk 个的方案数（恰好 kkk 个和至多 kkk 个均为某种条件）。

假设可以快速计算出 gkg_kgk​，我们要通过 gkg_kgk​ 计算 fkf_kfk​。

可以知道，gk=∑i=1k(ki)fig_k = \sum\limits_{i = 1}^k \dbinom ki f_igk​=i=1∑k​(ik​)fi​

（枚举恰好 iii 个满足条件，有 kkk 选 iii 种方案，加起来）

根据二项式反演可以得出：

fk=∑i=1k(−1)k−i(ki)gif_k = \sum_{i = 1}^k (-1)^{k - i} \dbinom ki g_ifk​=i=1∑k​(−1)k−i(ik​)gi​

恰好和至少的转换：

同理，设 fkf_kfk​ 为恰好 kkk 个的方案数，gkg_kgk​ 为至少 kkk 个的方案数。

有：

gk=∑i=kn(ki)fig_k = \sum_{i = k}^n \dbinom ki f_igk​=i=k∑n​(ik​)fi​

根据二项式反演：

fk=∑i=kn(−1)i−k(ki)gif_k = \sum_{i = k}^n (-1)^{i - k} \dbinom ki g_ifk​=i=k∑n​(−1)i−k(ik​)gi​

# 例题

# P4859 已经没有什么好害怕的了

首先对 a,ba, ba,b 从小到大排序。

设 dpi,jdp_{i, j}dpi,j​ 表示前 iii 个 aaa 中，至少有 jjj 对 a>ba > ba>b，考虑如何转移。

分两种情况:

  • aia_iai​ 对 jjj 没有贡献，那么加上 dp_

  • aia_iai​ 对 jjj 有贡献，那么加上 dpi−1,j−1×(pi−(j−1))dp_{i - 1, j - 1} \times (p_i - (j - 1))dpi−1,j−1​×(pi​−(j−1))。pip_ipi​ 为最小的 ai>ba_i > bai​>b 的 bbb 的位置，由于 bbb 是有序的，所以 pip_ipi​ 也是单调不降的。

写到一起：

dpi,j=dpi−1,j+dpi−1,j−1×(pi−(j−1))dp_{i, j} = dp_{i - 1, j} + dp_{i - 1, j - 1} \times (p_i - (j - 1))dpi,j​=dpi−1,j​+dpi−1,j−1​×(pi​−(j−1))

计算出 dpdpdp 数组之后。设 gig_igi​ 表示至少有 kkk 对 a>ba > ba>b 的方案数，即等于 dpn,k×(n−k)!dp_{n, k} \times (n - k)!dpn,k​×(n−k)!。

乘上 (n−k)!(n - k)!(n−k)! 是因为其他的可以随便排，只要保证至少有 kkk 对 a>ba > ba>b 即可。

再设 fkf_kfk​ 表示恰好 kkk 对 a>ba > ba>b 的方案数，那么有 gk=∑i=kn(ki)fig_k = \sum\limits_{i = k}^n \dbinom ki f_igk​=i=k∑n​(ik​)fi​

根据二项式定理：

fi=∑i=kn(−1)k−i(ki)gif_i = \sum_{i = k}^n (-1)^{k - i} \dbinom ki g_ifi​=i=k∑n​(−1)k−i(ik​)gi​

然后就可以计算出 fif_ifi​ 了。

需要注意的一个点，由于题目中要求的是 a>ba > ba>b 的对数减去 a<ba < ba<b 的对数为 kkk，所以 a>ba > ba>b 的对数为 n+k2\frac {n + k}22n+k​。

# 斯特林数

# 第一类斯特林数

[nm]{nm}{n \brack m} {n \brace m}[mn​]{mn​}

[nk]{n \brack k}[kn​] 表示将 nnn 个不同的数划分为 kkk 个圆排列的方案数。

递推式：

[nk]=[n−1k−1]+(n−1)[n−1k]{n \brack k} = {n - 1\brack k - 1} + (n - 1) {n - 1 \brack k}[kn​]=[k−1n−1​]+(n−1)[kn−1​]

第 nnn 个数单独一个圆排列，或者与前面的原排列合并到一起，放到任意一个数后面都行，所以乘上 (n−1)(n - 1)(n−1)。

# 第二类斯特林数

{nm}{n \brace m}{mn​} 表示把 nnn 个不同的数划分为 kkk 个不计顺序的非空子集的方案数。初值 [00]=0{0 \brack 0} = 0[00​]=0

递推式：

{nk}={n−1k−1}+k×{n−1k}{n \brace k} = {n - 1 \brace k - 1} + k \times {n - 1 \brace k}{kn​}={k−1n−1​}+k×{kn−1​}

第 nnn 个数单独在一个集合，或者 nnn 合并到之前的集合中，有 kkk 个集合。初值 {00}=1{0 \brace 0} = 1{00​}=1

第二类斯特林数较为常用，且有如下公式：

mn=∑k=0min(n,m){nk}(mk)k!m^n = \sum_{k = 0}^{min(n, m)} {n \brace k}\dbinom mk k!mn=k=0∑min(n,m)​{kn​}(km​)k!

证明：

考虑组合意义：

  • 等式左边为 nnn 个不同的球放到 mmm 个不同的盒子的方案数。

  • 右边：枚举非空盒子个数 kkk，先从 mmm 个盒子中选 kkk 个盒子，把 nnn 个不同的球球划分到 kkk 个盒子中，还要记盒子的顺序，所以再乘上 k!k!k!。

证明：

  • 左边：把 nnn 个球不同的球划分到 mmm 个盒子中。

  • 右边：容斥至少的空盒子个数，最后答案就是所有盒子非空的情况，盒子不考虑顺序，要除以 m!m!m!。

聪明伶俐的你一定发现了上面两个式子是符合二项式反演的！

简单推一下，先把第二个式子左右两边同乘 m!m!m!，并把等号右边的枚举空集合个数改成枚举非空集合，有：

{nm}m!=∑k=0m(−1)m−k(mm−k)kn{n \brace m}m! = \sum_{k = 0}^m (-1)^{m - k} \dbinom {m}{m - k}k^n{mn​}m!=k=0∑m​(−1)m−k(m−km​)kn

设 gk=kn,fk={nk}k!g_k = k^n, f_k = {n \brace k}k!gk​=kn,fk​={kn​}k!，容易看出：

gm=∑k=0m(mk)fkfm=∑k=0m(−1)m−k(mk)gkg_m = \sum_{k = 0}^m \dbinom mk f_k \\ f_m = \sum_{k = 0}^m(-1)^{m - k} \dbinom mk g_kgm​=k=0∑m​(km​)fk​fm​=k=0∑m​(−1)m−k(km​)gk​

正好就是二项式反演的式子。

n,m≤5000n, m \leq 5000n,m≤5000

首先不考虑列的限制，如果只有每行互不等价的方案数为 gm=(cm)n‾g_m = (c^m)^{\underline n}gm​=(cm)n​。

设如果有 mmm 列，且行列均互不等价的方案数为 fmf_mfm​。

gm=∑i=0m{mi}fig_m = \sum_{i = 0}^m {m \brace i} f_igm​=i=0∑m​{im​}fi​

然后就先计算出 gkg_kgk​，再套斯特林反演计算 fmf_mfm​ 即可。
